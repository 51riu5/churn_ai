MSTAN Architecture

Three novel components that make this architecture unique

Multi-Scale Temporal Convolutions

Parallel dilated causal convolutions at scales 1, 2, 4 capture short, medium, and long-range customer behavioural patterns simultaneously.

Gated Feature Fusion

A soft gating mechanism dynamically weights each temporal scale per customer. A slow contract downgrade benefits from long-range; a sudden support call from short-range.

Temporal Decay Attention

A learnable exponential decay bias -α|i-j| is added to attention logits, encoding the inductive bias that recent interactions are more predictive of churn.

Model Pipeline

Input (B, T, F) → Linear Proj → Multi-Scale Conv + Gate → Pos Encoding → [CLS] Prepend → N × Decay Attention → CLS Pool → Classifier
